FiDDL (Face iD Door Lock)

Currently In Development

(Deployment estimated to be fixed by Jan. 25)

This is a Personal Proposal senior design project, Flask web app that allows users to register an account and upload photos. These photos are stored in Google Firebase and used with OpenCV facial recognition to allow hands-free user access to their home using Google Nest Doorbell and Nuki Smart Lock API communication.
